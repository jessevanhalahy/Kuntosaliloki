# Kuntosaliloki
Tarkoituksena on tuottaa web-sovellus, johon käyttäjä voi rekisteröityä ja kirjata ylös omia kuntosalitreenejään ja kommentoida muiden suorituksia.
* Käyttäjä voi rekisteröityä ja kirjautua sovellukseen.
* Käyttäjä voi luoda, muokata ja poistaa kirjaamiaan kuntosalitreenien lokeja.
* Käyttäjä näkee sovelluksessa kirjatut lokit.
* Käyttäjä voi hakea eri treenien lokeja hakusanoilla.
* Treenejä voi hakea eri luokitteluilla (esim. cardio, painonnnosto).
* Käyttäjä voi jättää kommentteja omiin ja muiden käyttäjien lokeihin.
* Käyttäjillä on myös omat käyttäjäkohtaiset sivut, joissa näkyy kooste omista kirjatuista lokeista.
